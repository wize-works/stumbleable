# =============================================================================

# - Max 1 request per second per IP
# - Respect Cache-Control headers
# - Use If-Modified-Since headers
# - Identify your crawler in User-Agent

# ATTRIBUTION REQUIREMENTS

# 3. Respect user privacy - do not associate content with specific users
# 4. Do not scrape personal identifiable information (PII)
